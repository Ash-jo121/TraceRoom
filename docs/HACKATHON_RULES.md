# Hackathon Rules

TraceRoom is built for a SigNoz/Foundry-oriented hackathon submission.

Required:
- Integrate with SigNoz.
- Install SigNoz using Foundry.
- Include `casting.yaml`.
- Include `casting.yaml.lock` after generation.
- Use SigNoz MCP, Query Builder, dashboards, and alerts where possible.
- Disclose AI assistant usage.
- Do not use real-money trading.
- Do not store broker credentials.

The local demo must remain deterministic and must not depend on live trading systems.
